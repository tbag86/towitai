# towit.ai — Presentation (single file)

**One self-contained file: `index.html`.** Every image, logo and video is embedded inside it, so it
always displays — opened locally, emailed, AirDropped, or synced through iCloud.

## Present it
1. Double-click **`index.html`** — opens in any modern browser.
2. Press **`F`** for full-screen.
3. Navigate: **← / →** (or ↑ / ↓, space, scroll, swipe). Click the footer dots to jump.

## Don't edit this file directly
It's generated. Edit `../build/deck.html` and rebuild:
```bash
python3 ../build/build_inline.py
```
Full guide: `../CLAUDE.md`.
